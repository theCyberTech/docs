## Installing via pip

You can install crewAI using pip.
```bash
pip install crewai
```
and

```bash
pip install crewai-tools
```

## Installing via source

You can also install crewAI from source.

```bash
git clone https://github.com/crewAIInc/crewAI.git
cd crewAI
pip install -e .
```

and

```bash
git clone https://github.com/crewAIInc/crewAI-tools.git
cd crewAI-tools
pip install -e .
```

## Installing via Poetry

You can also install crewAI using Poetry.

```bash
poetry add crewai
  ```

and

```bash
poetry add crewai-tools
```
